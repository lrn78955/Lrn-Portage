# LRN PORTAGE

Site vitrine React/Vite pour LRN PORTAGE.

## Installation

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```

## Déploiement Vercel

1. Créer un dépôt GitHub
2. Pousser ce dossier sur GitHub
3. Importer le dépôt dans Vercel
4. Cliquer sur Deploy

Vercel détectera automatiquement Vite.
